# portfolioBackend

Thats uh, MongoDB, Express.js, Vue, and Node.js, for those of you uninformed.

## Project setup

```
npm i
```

### Compiles and hot-reloads for development

```
npm run start
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
